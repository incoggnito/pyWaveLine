## Install

```
git clone https://github.com/vallen-systems/pyWaveline
cd pyWaveline
pip install -e .[dev]
```

Run test suite:

```
tox
```

The documentation is built with [sphinx](https://www.sphinx-doc.org):

```
cd docs
sphinx-build . _build
```
